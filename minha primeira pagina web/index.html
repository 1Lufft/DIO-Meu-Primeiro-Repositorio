Como surgiu o HTML?

HTML sigla para HyperText Markup Language que em portugues significa Linguagem de Marcação de Hipertexto, foi criada por Tim Berners-Lee para ser uma ferramenta que facilitaria o dia-a-dia das empresas com o compartilhamento de documentos de forma mais pratica. Um ano depois foi criada a World Wide Web (WWW) que tornou o HTML uma ferramenta acessivel para o publico geral e se tornou a base do que conhecemos por web hoje.

Como é a estrutura do HTML?

A estrutura básica de um documento HTML (HyperText Markup Language) é organizada em uma série de elementos e tags que definem o conteúdo e a estrutura da página web. Aqui está um exemplo simples da estrutura básica de um documento HTML:

< html >

< head >

< title > < /title >

< head >

< body >

< /body >

< /html >

Algumas Tags do HTML.

Title: a tag usada para inserir o titulo da página

Body: tudo que esta contido dentro dessa tag é o que pode ser visível na página

P: é a tag que define um parágrafo
